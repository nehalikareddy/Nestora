Nestora – Full-Stack Accommodation Booking Website

🌐 Live Demo: https://nestora-ms4s.onrender.com/listings

Nestora is a full-stack accommodation booking web application inspired by modern rental platforms. It allows users to explore listings, create accounts, upload properties, and view locations on interactive maps.

🚀 Features

User authentication and authorization using Passport.js

Create, update, and delete property listings

Upload multiple images per listing using Cloudinary

Location-based mapping with Mapbox Geocoding API

Server-side rendering with EJS

MVC architecture with RESTful routing

Secure backend with middleware and session management

🛠 Tech Stack

Frontend: HTML, CSS, EJS
Backend: Node.js, Express.js
Database: MongoDB, Mongoose
Authentication: Passport.js
Cloud Storage: Cloudinary
Maps & Geocoding: Mapbox

📁 Project Structure (MVC Architecture)

Nestora follows the MVC design pattern:

models/ – Mongoose schemas and database logic

controllers/ – Business logic and request handling

routes/ – Express route definitions

views/ – EJS templates for server-side rendering

public/ – Static assets (CSS, JS, images)
